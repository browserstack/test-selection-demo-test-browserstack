# BrowserStack Smart Test Selection AI Agent
---
## What is Smart Test Selection?
BrowserStack's Smart Test Selection Agent uses AI to identify and run only the tests impacted (or likely to fail) by your code changes, reducing build time and cost by up to 50%. This demo uses the BrowserStack Java SDK with TestNG.

## Introduction
This repository demonstrates BrowserStack's Smart Selection AI Agent. For demo steps, checkout branch [`tesng-automate`](https://github.com/browserstack/test-selection-demo-test-browserstack/tree/testng-automate) and follow the readme
